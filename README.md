# Deepak Dabi Deals

A lightweight affiliate-deals website starter for GitHub Pages.

## Deploy
1. Upload all files to a GitHub repository.
2. Keep `index.html` in the repository root.
3. GitHub → Settings → Pages → Deploy from branch → `main` → `/root`.
4. Open the generated HTTPS URL.
5. Replace the placeholder product URLs in `app.js` with your own approved affiliate links.
6. Update `robots.txt` and `sitemap.xml` with the final GitHub Pages URL.
7. Add the live website URL to your Amazon Associates account.

## Important
- Do not publish fake prices, discounts, ratings or reviews.
- Use only product data, images and APIs permitted by your affiliate program.
- Follow Amazon Associates Program policies and required disclosure wording.
- The sample `https://www.amazon.in/` links are placeholders and are NOT affiliate links.
